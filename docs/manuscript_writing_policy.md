# Manuscript Writing Policy

The skill may plan manuscript structure and claim grounding. It must not write submission-ready manuscript prose unless the user separately provides real evidence and asks for writing.

Rules:

- Do not invent citations, author-year references, DOI values, datasets, numerical results, significance claims, reviewer comments, or conclusions.
- Do not convert a corpus-scoped gap into a field-wide novelty claim.
- Do not turn result placeholders into performance claims.
- Keep Introduction logic explicit: background -> streams -> gap -> problem -> contribution -> validation preview.
- Keep Related Work stream-based, not bibliography dumping.
- Link Method claims to problem properties and ablation requirements.
- Link Experiment claims to validation targets, baselines, metrics, ablations, statistics, and artifacts.
- Surface limitations and counterevidence before conclusion language.
